Version 4.0.8
=============

Released: October 1, 2021

Changes
-------

Reviewers
^^^^^^^^^

Added reviewers data and management tool.
A reviewer represents an admissions/recruitment officer who manages candidates, which are encapsulated in another new class.

Candidates
^^^^^^^^^^

Added candidates data and management tool.
A candidate represents some applicant/candidate for a position. It's geared towards student-applicants, with attributes like `undergraduate_school` and gpa.



Rename Docker entrypoint variable ``CELERY_TASKS_MEMORY_PER_CHILD_ARGUMENT``
to the correct ``CELERY_MAX_TASKS_PER_CHILD_ARGUMENT``.

Removals
--------

- None


Upgrade process
---------------

.. No special action required


Troubleshooting
---------------

If you observe the errors:

- ``mayan.apps.file_caching.models.CachePartitionFile.DoesNotExist: CachePartitionFile matching query does not exist.``

or:

- ``FileNotFoundError: [Errno 2] No such file or directory: '/var/lib/mayan/document_file_storage/...``

follow the solution outlined in the troubleshooting section: :ref:`troubleshooting-version-4.0`


Backward incompatible changes
-----------------------------

- Previous versions of mayan-edms are not compatable/will not recognize reviewer or candidate data.
    - Consider backing up all reviewer/candidate data before reverting to a previous version
